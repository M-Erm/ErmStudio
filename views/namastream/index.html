NamaStream

A Firefox new tab extension that replaces your browser's default new tab with a real-time dashboard of VTuber live streams and upcoming schedules, trying to mimic the firefox default new tab layout. Powered by the YouTube and Twitch APIs.

Backend built on Cloudflare Workers with CacheKV and Cron Triggers

Custom Wallpapers

Set custom wallpapers from URLs, upload from your PC, or rotate through saved collections.

It is possible to upload images, or just use the defaults. Everything syncs with your browser preferences.

  • Set wallpapers from URLs or local files
  • Save and rotate between multiple images
  • Image compression for better performance
  • Persistent storage across sessions

Custom Layout

In the popup it is possible to disable/enable certain channels, enable/disable bars, and tweak the layout, resizing / repositioning the bars

It is also possible to pin your favorite channels to appear first, Sort channels, et cetera.

  • Enable/disable individual channels
  • Sort by generation or manual order
  • Reposition stream bars
  • Resize panels for better viewing

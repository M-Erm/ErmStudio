# Portfolio

This repository contains the source code for my portfolio hosted on GitHub Pages

## Live

https://m-erm.github.io/Portfolio/

## Overview

The project is a single-page application built with vanilla JavaScript. It uses client-side routing based on URL paths to navigate between sections without reloading the page.

## Technologies

* HTML5
* CSS3
* Vanilla JS

## Features

* Single-page application architecture
* Client-side routing using URL paths
* Project and model organization through static assets
